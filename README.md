# Coca-Cola Sales Analysis

Exploratory data analysis and KPI dashboard project using Coca-Cola regional sales data. The goal was to clean raw sales data, uncover regional performance patterns, and build a visual report to support strategic sales decisions.

---

## The Business Problem

Regional sales data was unstructured and siloed — there was no clear view of which territories were growing, which were stagnating, and what was driving revenue concentration across the portfolio.

---

## What I Did

### 1. Data Cleaning
- Removed duplicates, handled missing values, and standardised column formats
- Normalised currency fields and date formats for consistent analysis

### 2. Exploratory Data Analysis
- Analysed revenue distribution across regions
- Identified top and bottom performing territories
- Examined sales trends and seasonal patterns

### 3. KPI Dashboard (Power BI)
Built an interactive Power BI dashboard tracking:
- Revenue by region
- Year-on-year growth rates
- Top SKUs and product categories
- Territory performance ranking
